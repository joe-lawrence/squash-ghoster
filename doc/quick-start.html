Getting Started

The Squash Ghoster tool is designed to help you create structured, timed workouts for squash training. Whether you're a coach looking to create drills for your players or a player wanting to practice specific shot patterns, this tool provides the flexibility to design workouts that match your training goals.

The application works by allowing you to define shot patterns, set timing intervals, and configure audio cues that will guide you through your workout. You can create simple repetitive drills or complex multi-pattern sequences with varying difficulty levels and timing requirements.

Workout Structure

Squash Ghoster organizes workouts using a hierarchical structure: Workout → Pattern → Shot/Message. A workout contains one or more patterns, each pattern contains multiple shots and messages, and each shot or message has its own timing and audio settings. This structure allows you to create complex training routines by combining simple building blocks.

Shot Sequence Timing

Each shot in a workout follows a timing sequence designed to facilitate ghosting practice. The sequence includes narration (with lead time for recognition), a split-step cue, and a final shot sound. This structured timing helps you develop consistent movement patterns, footwork, and shot preparation techniques essential for effective ghosting training.

Example Shot Cycle Timeline (4.0 seconds)

0.0s: Shot period begins
1.5s: Narration begins (2.5s ahead of shot)
3.7s: Split step sound plays
4.0s: Shot sound plays & period ends

Open the Application

Navigate to the main application page to begin creating your squash workout. The interface will load with a default workout configuration that you can modify or replace entirely.

Load a Sample Workout

Start by loading one of the included sample workouts to see how the system works. These examples demonstrate different approaches to workout design and will help you understand the various features available.

Explore the Interface

Familiarize yourself with the floating toolbar and main interface elements. The toolbar provides quick access to all major functions, while the main area shows your current workout configuration.

Main Menu Mode

The main menu mode is your primary workspace for configuring squash workouts. This is where you'll spend most of your time setting up patterns, shots, and timing. The interface is designed to be intuitive while providing powerful customization options for creating workouts that match your specific training needs.

In this mode, you can define the structure of your workout by creating patterns that contain specific shot sequences. Each pattern can be repeated multiple times, shuffled for variety, or combined with other patterns to create complex training routines. The system supports both simple repetitive drills and sophisticated multi-pattern workouts with varying timing and difficulty levels.

Key Features

  • Workout Configuration: Set up patterns, shots, and timing intervals with precise control over each element
  • Pattern Management: Create and organize shot patterns with repeats, shuffling, and conditional logic
  • Timing Controls: Configure shot intervals, prep times, and message timing for optimal training flow
  • Audio Settings: Set up text-to-speech voices, audio ducking, and sound effect preferences

The Floating Toolbar

The floating toolbar is your command center, providing quick access to all major functions. It's positioned in the top-left corner and is collapsed by default to keep your workspace clean. The gear icon is always visible, and selecting it will expand the toolbar as depicted in the screenshot above.

The toolbar is designed to be unobtrusive while remaining easily accessible. It automatically hides when you click outside of it, but you can bring it back by clicking the gear icon or using keyboard shortcuts for power users.

Each button in the toolbar opens a dropdown menu with related functions, allowing you to quickly perform common tasks like loading workouts, saving your progress, or starting a workout session.

Toolbar Buttons

  • Editor: Access the advanced workout editor for detailed configuration
  • Execute: Run, preview, or set prep time for your workouts
  • Load: Load workout from file or clipboard
  • Save: Save workout to file or clipboard
  • Favorites: Manage favorite workouts for quick access
  • Audio Ducking: Toggle audio ducking mode to manually keep the Text To Speech engine active, preventing the OS from constantly adjusting system sound levels up/down during a workout
  • Rocket Surgery Mode: Enable advanced features for power users
  • Theme: Select light or dark color schemes
  • Help: Access documentation and support resources

Preview Mode

Preview mode lets you see exactly how your workout will play out before you run it. It shows the complete timeline with all shots, messages, and timing, giving you the opportunity to review and adjust your workout configuration before committing to a full training session. This is particularly useful for complex workouts with multiple patterns or when you want to verify timing calculations.

The preview interface provides a comprehensive overview of your workout, including total duration, shot counts, and pattern breakdowns. You can play through the entire timeline to hear how the audio cues will sound, check timing accuracy, and identify any potential issues with your workout design. This helps ensure that your training session will flow smoothly and meet your intended training objectives.

How to Access Preview

Click the Execute button in the floating toolbar

Select Preview from the dropdown menu

The preview modal will open showing your workout timeline

Preview Features

  • Timeline Playback: Play through the workout timeline to hear audio cues and verify timing accuracy
  • Time Navigation: Jump to any point in the workout timeline for quick review of specific sections
  • Workout Summary: See total duration, shot count, and pattern breakdown for planning purposes
  • Presentation Mode: Jump directly to presentation mode from any point in the timeline

Presentation Mode

Presentation mode is the full-screen workout execution interface. This is where you'll actually run your squash workout with audio cues and visual feedback. The interface is designed to be distraction-free, allowing you to focus entirely on your training while receiving clear guidance through audio announcements and visual progress indicators.

When you enter presentation mode, the application transforms into a dedicated training environment. The interface displays your current shot, progress through the workout, and provides real-time feedback on your performance. The system is designed to work seamlessly with your training routine, whether you're practicing alone or with a coach providing additional guidance.

How to Access Presentation Mode

Click the Execute button in the floating toolbar

Select Run from the dropdown menu

Or use the Present button in preview mode to jump to a specific time

Presentation Mode Controls

Once in presentation mode, use the Present button in the top right to begin or pause the workout. The Close button (also in the top right) will return you to the main configuration mode.

Presentation Mode Features

  • Audio Cues: Text-to-speech announcements for each shot and message with customizable voice settings
  • Progress Tracking: Visual progress bar and shot counter to monitor your advancement through the workout
  • Playback Controls: Pause, resume, and skip functionality for flexible training sessions
  • Visual Feedback: Animated ghost indicator showing workout status and providing visual motivation
  • Rocket Mode: Advanced timing visualization with tick marks for precise timing control

Loading and Saving Workouts

The application supports multiple ways to load and save your workout configurations. You can work with files, clipboard, and favorites, making it easy to share workouts with others, backup your configurations, or quickly switch between different training routines. The system includes built-in validation to ensure that loaded workouts are properly formatted and will work correctly.

All workout data is stored in JSON format, which is both human-readable and easily shareable. This means you can edit workout files in any text editor if needed, or share them with other players and coaches. The application will automatically validate any loaded workout to ensure it contains all required elements and follows the proper structure.

Pro Tip

Use the clipboard features to quickly share workouts with others or backup your configurations. The JSON format is human-readable and can be easily edited in any text editor. This makes it easy to create variations of existing workouts or troubleshoot any issues with your workout configuration.

Next Steps

Now that you understand the basics of the Squash Ghoster tool, you're ready to start creating and running your own workouts. The following steps will help you progress from a beginner to an advanced user, gradually exploring all the features and capabilities that the system has to offer.

Try a Sample Workout

Load one of the included sample workouts and run it in presentation mode to see how everything works. This will give you a feel for the timing and audio cues before you start creating your own workouts.

Create Your First Workout

Start with the default workout and modify it to create your own custom patterns and timing. Begin with simple changes like adjusting shot intervals or adding new shots to existing patterns.

Try the editor mode, experiment with audio ducking, and explore rocket mode for advanced timing visualization. These features will help you create more sophisticated and effective training routines.

Share Your Workouts

Save your creations and share them with other players using the clipboard or file export features. This allows you to build a library of workouts and collaborate with other players and coaches.
